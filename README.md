# Behavioral Signal Lab (MistralAI SF Hackathon)

AI-powered behavioral pattern analysis from sparse conversational text.

## Overview

Behavioral Signal Lab evaluates how well large language models can infer:

- Decision strategy archetypes
- Conflict response mechanisms
- Risk calibration patterns
- Emotional regulation signals
- Stress response shifts
- Internal behavioral tensions

The system avoids:
- Demographic inference
- Mental health diagnosis
- Protected trait speculation

It focuses purely on observable communication signals.

## Research Question

How reliably can open-weight models extract structural behavioral patterns from limited conversational input?

## Method

- Structured behavioral prompt
- Linguistic justification required
- Evidence quotes mandatory
- Uncertainty estimation included
- Reliability heuristic based on sample size

## Evaluation Dimensions

We evaluate model outputs on:

1. Structural Depth
2. Linguistic Justification
3. Internal Tension Detection
4. Uncertainty Calibration
5. Guardrail Compliance

## Limitations

- Small text samples reduce reliability
- Inference quality depends on signal density
- Non-diagnostic by design

## Demo Flow

1. Paste 3–6 short responses
2. Click Analyze
3. Review structured behavioral memo
4. Observe uncertainty + reliability score
