# Judge Script (2–3 minutes)

1) Start session.
2) Round 1: answer naturally for ~4 turns.
3) Click "Advance round" to enter Round 2.
4) Round 2: try to hide / randomize for ~4 turns.
5) Click "Advance round" to finish.
6) Reveal report.

Talking point:
- "We measure how hard it is for humans to avoid being modeled by an adaptive agent."
- "We only infer behavioral signals, not sensitive attributes."
